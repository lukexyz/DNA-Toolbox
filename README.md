## Installation (`WSL/Ubuntu`)

```sh
git clone https://github.com/lukexyz/DNA-Toolbox.git
conda create -n dna-toolbox python=3.6 pip jupyter
conda activate dna-toolbox
pip install -r requirements.txt
```
